# The Archer Group

Static website for [The Archer Group](https://www.archergroup.co) — practical nonprofit management and fundraising consulting, based in San Francisco.

Built with plain HTML and CSS, hosted on GitHub Pages.

## Pages

- `index.html` — Home
- `about.html` — About / founder bio
- `services.html` — Consulting services
- `testimonials.html` — Client testimonials
- `contact.html` — Contact

## Local preview

Any static file server works, e.g.:

```
python3 -m http.server 4173
```

then open http://localhost:4173
